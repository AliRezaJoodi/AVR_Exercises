## Config Timer0 in PWM Mode

MCU:        ATmega32  
Frequency:  8MHz  

The PWM frequency for the output can be calculated by the following equation:

$$ FrequencyForPWM(OC0) = {clk \over Prescaler * 256 } $$

Prescaler: 1, 8, 64, 256 or 1024
